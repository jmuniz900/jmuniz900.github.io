Spotify API

During college, I really wanted to learn how to use an API. This lead to me creating a Spotify Music Recommender using the Spotify API to allow a user to input a song and get a request from the API to output ten more songs in the same genre as the song inputted. It's a very crude program in the form it's in and definitely still needs improvements to both the front-end and back-end.

Personal Website
